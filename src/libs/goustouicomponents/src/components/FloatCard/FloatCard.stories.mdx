# FloatCard

FloatCard should show as floating in the bottom left corner of the screen.

If you want the `FloatCard` to be closer or further away from the bottom of the window, you can do that by giving the `offsetVertical` a CSS offset value (e.g. `5rem`). Check the `defaultProps` for the default value given to this.

If you want to pass a custom function when clicking on the close icon, you can do it by passing that function to the `onCloseIconClick` prop.

## closeIcon

* `true` - always showing the close icon
* `false` - never showing the close icon
* `small-screens-only` - will have the close icon only on smaller screens
