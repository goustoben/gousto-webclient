# VerticalStagesItem

This component should be used as a child of the VerticalStages component. It is therefore assumed that certain props will be:

1. Required here but only used by VerticalStagesItem used within VerticalStages
2. Passed down from VerticalStages

If you want to add extra styling to the content wrapper inside VerticalStagesItem, you can pass a className using the `extraClass` prop.

`hasFullWidth` affects only the last step by allowing the content to go below the numbered element.
