# LayoutContentWrapper

This layout component is a container that helps wrap the content and adds padding to it. It allows for the disabling of  the horizontal and/or vertical padding, if only one/none is/are needed.

The `large/xlarge` value allows for a degree of responsiveness: on the mobile screen sizes, the padding is 1rem, and on tablet screen sizes and above, the padding is 2rem.
