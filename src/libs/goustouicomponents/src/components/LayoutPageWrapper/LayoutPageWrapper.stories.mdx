# LayoutPageWrapper

This layout component is a container that helps to keep the content on the page in a consistent wrapper. If offers a predefined left/right padding (that can be disabled in certain cases - see props) and a max width, allowing the developers to control the spread of the information on very large screens.

## Padding on larger screens:
The padding is only applied on large screens. On mobile, the left and right paddings are set to 0.

## Expose variables from the Layout configuration
When Zest users want to use this component they may want to align in terms of paddings, margins, etc.
If a child components of this needs to adapt to the layout of this component there are two exposed CSS variables that can be used to drive those changes:
- `LayoutPageWrapperDesktopScreenBreakpointMin`: This is the breakpoint in which padding is applied when `padding` prop is `large-screens-only`
- `LayoutPageWrapperPadding`: This is the value of the padding applied

### Example of use
- In Zest:
```
@value LayoutPageWrapperPadding, LayoutPageWrapperDesktopScreenBreakpointMin from 'design-language/utils.module.css';
```
- In a project that uses Zest:
```
@value LayoutPageWrapperPadding, LayoutPageWrapperDesktopScreenBreakpointMin from '~goustouicomponents/dist/design-language/utils.module.css';
```
