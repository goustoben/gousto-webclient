# Header

This is an MVP implementation of the header for Gousto. For now, it will not handle anything related to authentication (login/logout).

The menu items are hardcoded and read from `gousto-config/routes.js`. If you want to change the items rendered, the component has them listed at the top of the file. The naming must match the one from the routes file.
