# CollectionsNavigation
 This component expects to have `CollectionsNavigationItem` as children. Each `CollectionsNavigationItem` has a required `onClick` prop that is a function that can be passed to control what happens when you click the item. If by default you want one of the `CollectionsNavigationItem` to be active when `CollectionsNavigation` renders, you can pass the `isActive` prop as a boolean.

## Props for CollectionsNavigation

</Alert>
